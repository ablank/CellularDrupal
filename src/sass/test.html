Cellular UI Javascript Components

jAccordion

Convert an HTML list into an accordion style widget.

  • Markup

    <ul class="jAccordion">
<li>
    <p>Title 1</p>
    <div>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
    </div>
</li>
<li>
    <p>Title 2</p>
    <p>Pellentesque egestas mi nec odio convallis dapibus.</p>
</li>
<li>
    <p>Title 3</p>
    <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p>
</li>
</ul>

  • JS

    <script>
$('.selector').jAccordion({
        "active": 0, // Index value of initial content to display.
        "duration": 500, // Duration of transition.
        "easing": "swing", // Type of easing.
        "single": false // Allow multiple panels to be opened or only 1?
    });
</script>

  • CSS

jCard

Wrap a block of content with a link.

  • Markup

    <ul class="jBlocklink">
<li>
    <p>Title 1</p>
    <div>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
    </div>
</li>
<li>
    <p>Title 2</p>
    <p>Pellentesque egestas mi nec odio convallis dapibus.</p>
</li>
<li>
    <p>Title 3</p>
    <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p>
</li>
</ul>

  • JS

    <script>
$('.selector').jBlocklink({
  "cclass": "jBlocklink-link" // Class to add to the link for styling.
});
</script>

  • CSS

jMmenu

Mobile friendly menu, adds menu button if window size is less than the set breakpoint.

  • Markup

    <ul class="jMmenu">
<li><a href="#">Link</a></li>
<li><a href="#">Link</a></li>
<li><a href="#">Link</a></li>
</ul>

  • JS

    <script>
$('.selector').jMmenu({ 
  // Window breakpoint trigger, in pixels
  "breakpoint": 650,
  // Classes combined & added to <body> for styling.page elements.
  "type": "slide",
  "direction": "right"
});
</script>

  • CSS

jScrolli

Rotate display through a list of content.

  • Markup

    <ul class="jScrolli">
<li>
    <p>Title 1</p>
    <div>
    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. </p>
    </div>
</li>
<li>
    <p>Pellentesque egestas mi nec odio convallis dapibus.</p>
</li>
<li>
    <p> Nullam eget arcu vulputate, tincidunt dolor quis, bibendum ex.</p>
</li>
</ul>

  • JS

    <script>
$('.selector').jScrolli({
  "speed": 500, // Duration of cycle
  "pause": 3000 // Time to pause between cycles
});
</script>
